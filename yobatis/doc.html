T selectOne(PK pk);

根据主键查询记录。
返回与表对应的对象如果主键存在，否则NULL。

T selectOne(BaseCriteria criteria);

根据条件查询一条记录。构建criteria时，调用setForUpdate(true)方法可以加入for update子句。
返回对应的对象如果存在，否则NULL；如果有多条记录满足查询条件，则抛出异常。

List selectList(BaseCriteria criteria);

查询满足条件criteria的记录。构建criteria时，调用setForUpdate(true)方法加入for update子句, setLimit()方法加入limit, setOffset()方法加入offset。
返回对应的对象如果存在，否则空List。

long count(BaseCriteria criteria);

统计满足条件criteria的记录数量。
返回记录数量。

long countAll();

统计表中所有记录数量。
返回记录数量。

int insertAll(B record);

插入record中的所有字段。
插入成功返回1；失败则抛出相关异常。

int insertAllIgnore(B record);

插入record中的所有字段，该方法不会抛出异常如果插入失败。
插入成功返回1；失败则返回0。

int insert(B record);

插入record中的所有非NULL字段；若表为自增主键且record中的主键为NULL，则会在插入后将新生成的主键设置到record中对应的字段。
插入成功返回1；失败则抛出相关异常。

int update(B record);

基于主键更新record中的所有非NULL字段。
更新成功返回1；如果没有对应记录则返回0。

int updateAll(B record);

基于主键更新record中的所有字段。
更新成功返回1；如果没有对应记录则返回0。

int update(B record, BaseCriteria criteria);

根据查询条件criteria更新对应记录，只更新record中非NULL字段。
返回被影响的记录数。

int updateAll(B record, BaseCriteria criteria);

根据查询条件criteria更新对应记录，更新record中所有字段。
返回被影响的记录数。

int delete(PK pk);

删除主键pk对应的记录。
返回1如果存在对应的记录，否则0。

int delete(BaseCriteria criteria);

删除查询条件criteria对应的记录。
返回被删除的记录数。
